# HuHoBot EndStone Adapter 项目架构文档

## 项目概述

HuHoBot EndStone Adapter 是专为EndStone设计的下一代基岩版服务器管理解决方案，提供安全的无第三方QQ机器人依赖管理体验。通过WebSocket协议实现服务器与控制端的即时绑定，基于事件驱动的非侵入式通信协议。

## 核心架构

### 1. 插件主类 (HuHoBot)

**职责**：
- 作为EndStone插件的入口点
- 管理插件生命周期（onLoad, onEnable）
- 处理玩家聊天事件和命令
- 协调各个组件的初始化和交互

**关键功能**：
- 广播消息到服务器
- 执行服务器命令
- 获取在线玩家列表
- 设置定时任务（重连、心跳、自动断连）

### 2. WebSocket客户端 (WebSocketClient)

**职责**：
- 提供WebSocket连接的底层实现
- 处理WebSocket协议的握手、数据帧解析
- 管理连接状态（Open, Closing, Closed）

**核心特性**：
- 跨平台支持（Windows/Linux）
- 支持文本和二进制数据传输
- 自动处理Ping/Pong心跳
- 线程安全的消息发送
- 连接状态管理

**关键方法**：
- `Connect()`: 建立WebSocket连接
- `SendText()`/`SendBinary()`: 发送数据
- `OnTextReceived()`/`OnBinaryReceived()`: 注册接收回调
- `Ping()`/`Pong()`: 心跳机制
- `Close()`/`Shutdown()`: 关闭连接

### 3. 机器人客户端 (BotClient)

**职责**：
- 实现业务逻辑层
- 处理与服务器的通信协议
- 管理服务器绑定和认证
- 处理各种事件（聊天、命令、白名单等）

**核心功能**：
- 握手认证机制
- 心跳保持
- 自动重连
- 消息路由和事件处理
- 配置同步

**事件处理**：
- `sendConfig`: 配置同步
- `chat`: 聊天消息处理
- `add`/`delete_`: 白名单管理
- `cmd`: 命令执行
- `queryList`/`queryOnline`: 查询功能
- `run`/`runAdmin`: 自定义命令执行
- `heart`: 心跳响应
- `bindRequest`/`bindConfirm`: 绑定流程

### 4. 配置管理器 (ConfigManager)

**职责**：
- 管理插件配置文件
- 提供配置读写接口
- 支持配置热重载
- 版本管理和配置迁移

**配置项**：
- `serverId`: 服务器唯一标识
- `hashKey`: 认证密钥
- `chatFormat`: 聊天格式配置
- `motdUrl`: 服务器信息显示URL
- `serverName`: 服务器名称
- `customCommand`: 自定义命令配置
- `callbackConvertImg`: 回调转换图片配置

### 5. 工具函数 (tools)

**职责**：
- 提供通用工具函数
- 数据处理和转换

**功能**：
- `generate_pack_id()`: 生成唯一包ID
- `splitString()`: 字符串分割
- `filterSet()`: 集合过滤

## 通信协议

### 消息格式

所有WebSocket消息采用JSON格式：

```json
{
  "header": {
    "type": "事件类型",
    "id": "消息ID"
  },
  "body": {
    // 事件特定的数据
  }
}
```

### 事件类型

**服务器发送事件 (ServerSendEvent)**：
- `sendMsg`: 发送消息
- `heart`: 心跳
- `chat`: 聊天
- `success`: 成功响应
- `error`: 错误响应
- `shakeHand`: 握手
- `queryWl`: 查询白名单
- `queryOnline`: 查询在线
- `bindConfirm`: 绑定确认

**服务器接收事件 (ServerRecvEvent)**：
- `sendConfig`: 配置同步
- `shaked`: 握手完成
- `chat`: 聊天消息
- `add`: 添加白名单
- `delete`: 删除白名单
- `cmd`: 执行命令
- `queryList`: 查询列表
- `queryOnline`: 查询在线
- `shutdown`: 关闭连接
- `run`: 执行自定义命令
- `runAdmin`: 管理员执行
- `heart`: 心跳响应
- `bindRequest`: 绑定请求

## 构建系统

### CMake配置

**关键配置**：
- C++20标准
- EndStone框架集成
- WebSocket服务器URL可配置
- 自动生成配置头文件

**构建命令**：
```bash
cmake -B build -DHUHOBOT_SERVER_URL=ws://your-server:port
cmake --build build
```

## 总结

HuHoBot EndStone Adapter采用模块化设计，通过WebSocket实现高效的服务器管理。核心架构包括：

1. **插件层**：与EndStone框架集成
2. **业务层**：处理游戏逻辑和命令
3. **通信层**：WebSocket连接管理
4. **配置层**：配置管理和持久化

通过心跳、重连、状态管理等机制确保连接稳定性，并提供了多种优化方案以提升性能和可靠性。
