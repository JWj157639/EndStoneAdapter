# 账号设置功能说明

## 概述

账号设置模块允许主人管理机器人账号的各种信息，包括昵称、头像、签名和性别等。该模块仅限主人使用，提供了便捷的命令来快速更新机器人资料。

## 功能特性

- ✅ 更改机器人昵称
- ✅ 更改机器人头像（支持图片）
- ✅ 更改机器人签名
- ✅ 更改机器人性别（男/女/未知）
- ✅ 仅限主人权限
- ✅ 仅支持群聊操作

### 更改机器人信息

#### 更改机器人昵称
- **权限**: OWNER（仅限主人）
- **用法**: `更改机器人昵称[内容]`
- **参数**:
  - `[内容]`: 新的昵称（不能为空）
- **示例**:
  ```
  用户: 更改机器人昵称小助手
  机器人: 好哒，已经把昵称改成这个内容啦！你看看我主页叭！
  ```

#### 更改机器人头像
- **权限**: OWNER（仅限主人）
- **用法**: `更改机器人头像[图片]`
- **参数**:
  - `[图片]`: 图片文件（必须附带在消息中）
- **示例**:
  ```
  用户: [发送图片] 更改机器人头像
  机器人: 我马上就去把头像改成介个，等我一会～～～
          [CQ:image,file=图片URL]
  ```
- **注意**: 必须在消息中附带图片，否则会提示"请携带一张照片再发送哟～"

#### 更改机器人签名
- **权限**: OWNER（仅限主人）
- **用法**: `更改机器人签名[内容]`
- **参数**:
  - `[内容]`: 新的签名（不能为空）
- **示例**:
  ```
  用户: 更改机器人签名今天也要加油鸭！
  机器人: 好哒，已经把签名改成这个内容啦！你看看我主页叭！
  ```

#### 更改机器人性别
- **权限**: OWNER（仅限主人）
- **用法**: `更改机器人性别[男|女]`
- **参数**:
  - `[男|女]`: 性别选项（男、女、未知）
- **示例**:
  ```
  用户: 更改机器人性别女
  机器人: 好哒，已经把性别改成【女】

  用户: 更改机器人性别男
  机器人: 好哒，已经把性别改成【男】

  用户: 更改机器人性别未知
  机器人: 好哒，已经把性别改成【未知】
  ```

## 使用限制

### 权限限制
- 仅限主人使用
- 非主人用户无法执行这些操作

### 消息类型限制
- 仅支持群聊消息
- 私聊时会提示"该功能仅支持群聊使用哦～"

### 参数验证
- 昵称/签名：内容不能为空
- 头像：必须附带图片
- 性别：仅支持"男"、"女"、"未知"

## API接口

### 使用的NapCat API

#### get_stranger_info
- **用途**: 获取机器人当前信息
- **参数**:
  ```typescript
  {
    user_id: number  // 机器人QQ号
  }
  ```
- **返回值**:
  ```typescript
  {
    user_id: number,
    nickname: string,  // 当前昵称
    sex: number,       // 当前性别（0=未知，1=男，2=女）
    // ... 其他字段
  }
  ```

#### set_qq_avatar
- **用途**: 设置机器人头像
- **来源**: NapCat扩展API
- **参数**:
  ```typescript
  {
    file: string  // 图片URL或文件路径
  }
  ```
- **示例**:
  ```typescript
  await ctx.actions.call(
    'set_qq_avatar',
    { file: 'https://example.com/avatar.jpg' },
    ctx.adapterName,
    ctx.pluginManager.config
  );
  ```

#### set_qq_profile
- **用途**: 设置机器人个人资料
- **来源**: go-cqhttp兼容API
- **参数**:
  ```typescript
  {
    nickname: string,      // 昵称
    personal_note: string, // 签名
    sex: number            // 性别（0=未知，1=男，2=女）
  }
  ```
- **示例**:
  ```typescript
  // 更改昵称
  await ctx.actions.call(
    'set_qq_profile',
    { nickname: '小助手' },
    ctx.adapterName,
    ctx.pluginManager.config
  );

  // 更改签名
  await ctx.actions.call(
    'set_qq_profile',
    { 
      nickname: '小助手',
      personal_note: '今天也要加油鸭！'
    },
    ctx.adapterName,
    ctx.pluginManager.config
  );

  // 更改性别
  await ctx.actions.call(
    'set_qq_profile',
    { 
      nickname: '小助手',
      sex: 2  // 女
    },
    ctx.adapterName,
    ctx.pluginManager.config
  );
  ```

function extractImages(message: OB11Message): string[] {
  const images: string[] = [];
  
  if (Array.isArray(message.message)) {
    message.message.forEach((item: any) => {
      if (item.type === 'image' && item.data) {
        const url = item.data.url || item.data.file;
        if (url) images.push(url);
      }
    });
  }
  
  return images;
}

#### 3. 更改机器人信息

```typescript
async function handleUpdateAccount(
  message: string,
  event: OB11Message,
  ctx: NapCatPluginContext,
  parsed: Record<string, string>
): Promise<void> {
  // 仅支持群聊
  if (event.message_type !== 'group') {
    await sendReplyMessage(ctx, event, '该功能仅支持群聊使用哦～');
    return;
  }

  const type = parsed.group0 || ''; // 头像、昵称、性别、签名
  const value = parsed.group1 || ''; // 具体内容

  // 获取当前机器人信息
  const currentInfo = await ctx.actions.call(
    'get_stranger_info',
    { user_id: event.self_id },
    ctx.adapterName,
    ctx.pluginManager.config
  );

  if (!currentInfo) {
    await sendReplyMessage(ctx, event, '获取机器人信息失败！');
    return;
  }

  // 处理头像
  if (type === '头像') {
    const images = extractImages(event);
    if (images.length === 0) {
      await sendReplyMessage(ctx, event, '请携带一张照片再发送哟～');
      return;
    }

    await ctx.actions.call(
      'set_qq_avatar',
      { file: images[0] },
      ctx.adapterName,
      ctx.pluginManager.config
    );
    
    let 回复 = '我马上就去把头像改成介个，等我一会～～～';
    回复 += `\n[CQ:image,file=${images[0]}]`;
    await sendReplyMessage(ctx, event, 回复);
    return;
  }

  // 处理昵称或签名
  if (type === '昵称' || type === '签名') {
    if (!value || value.trim() === '') {
      await sendReplyMessage(ctx, event, `更改${type}时需要填内容哦～`);
      return;
    }

    const params: any = {
      nickname: currentInfo.nick || ''
    };

    if (type === '昵称') {
      params.nickname = value;
    } else if (type === '签名') {
      params.personal_note = value;
    }

    await ctx.actions.call(
      'set_qq_profile',
      params,
      ctx.adapterName,
      ctx.pluginManager.config
    );

    await sendReplyMessage(ctx, event, `好哒，已经把${type}改成这个内容啦！你看看我主页叭！`);
    return;
  }

  // 处理性别
  if (type === '性别') {
    const genderMap: Record<string, number> = {
      '男': 1,
      '女': 2,
      '未知': 0
    };

    const genderValue = genderMap[value] ?? 0;
    const genderName: Record<number, string> = {
      0: '未知',
      1: '男',
      2: '女'
    };

    await ctx.actions.call(
      'set_qq_profile',
      {
        nickname: currentInfo.nick || '',
        sex: genderValue
      },
      ctx.adapterName,
      ctx.pluginManager.config
    );

    await sendReplyMessage(ctx, event, `好哒，已经把性别改成【${genderName[genderValue]}】`);
    return;
  }

  await sendReplyMessage(ctx, event, '不支持的操作类型！');
}
```

```typescript

const value = match[2]; // 具体内容
```

## 注意事项

1. **权限要求**: 仅限主人使用
2. **群聊限制**: 仅支持群聊操作，私聊不可用
3. **头像要求**: 必须附带图片，不支持文字描述
4. **参数验证**: 昵称和签名内容不能为空
5. **性别限制**: 仅支持"男"、"女"、"未知"三个选项
6. **API依赖**: 依赖NapCat的API接口，确保NapCat版本支持这些功能

## 故障排除

### 问题1：更改头像失败
- **原因**: 未附带图片或图片URL无效
- **解决**: 确保消息中包含图片，且图片URL可访问

### 问题2：获取机器人信息失败
- **原因**: NapCat API调用失败
- **解决**: 检查NapCat日志，确认API是否正常工作

### 问题3：更改性别无效
- **原因**: 性别参数不正确
- **解决**: 确保使用"男"、"女"、"未知"三个选项之一

### 问题4：私聊无法使用
- **原因**: 功能仅支持群聊
- **解决**: 在群聊中执行指令
