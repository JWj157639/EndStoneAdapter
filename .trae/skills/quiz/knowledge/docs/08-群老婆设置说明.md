# 群老婆设置功能说明

## 概述

群老婆设置模块是一个娱乐功能，允许用户随机抽取群友作为"今日老婆"。系统支持单群模式和全群模式，可以配置冷却时间、过滤机器人等选项。

## 功能特性

- ✅ 随机抽取群友作为老婆
- ✅ 支持单群模式（仅当前群）
- ✅ 支持全群模式（所有群）
- ✅ 可配置冷却时间
- ✅ 可过滤机器人账号
- ✅ 可过滤本人和机器人
- ✅ WebUI管理界面
- ✅ 支持更换老婆
- ✅ 历史记录避免重复

## 指令列表

### 菜单类

#### 群老婆
- **权限**: USER（需要卡密授权）
- **用法**: `群老婆`
- **描述**: 显示群老婆功能菜单
- **示例**:
  ```
  用户: 群老婆
  机器人: ══════════════
          普通指令↓
          - 今日老婆
          - 更换老婆
          
          管理员指令↓
          - 更改群老婆范围全群模式
          - 更改群老婆范围单群模式
          - 更改群老婆冷却[时长:秒]
          - [开启|关闭]群老婆过滤官机
          - [开启|关闭]群老婆过滤本人
          ══════════════
  ```

### 抽取类

#### 今日老婆/我的老婆
- **权限**: USER
- **用法**: `今日老婆` 或 `我的老婆`
- **描述**: 随机抽取一个群友作为今日老婆
- **注意**: 每天只能抽取一次，如需更换请使用"更换老婆"指令

#### 更换老婆/换群老婆
- **权限**: USER
- **用法**: `更换老婆` 或 `换群老婆`
- **描述**: 更换今日老婆（受冷却时间限制）
- **冷却**: 默认30秒，可通过管理员指令修改

### 管理类（仅限主人）

#### 更改群老婆范围全群模式
- **权限**: OWNER（仅限主人）
- **用法**: `更改群老婆范围全群模式`
- **描述**: 设置为全群模式，从所有群的成员中抽取

#### 更改群老婆范围单群模式
- **权限**: OWNER
- **用法**: `更改群老婆范围单群模式`
- **描述**: 设置为单群模式，仅从当前群成员中抽取（默认）

#### 更改群老婆冷却[时长]
- **权限**: OWNER
- **用法**: `更改群老婆冷却[秒数]`
- **示例**: `更改群老婆冷却60`
- **描述**: 设置更换老婆的冷却时间（秒）

#### 开启群老婆过滤官机
- **权限**: OWNER
- **用法**: `开启群老婆过滤官机`
- **描述**: 开启过滤机器人账号

#### 关闭群老婆过滤官机
- **权限**: OWNER
- **用法**: `关闭群老婆过滤官机`
- **描述**: 关闭过滤机器人账号

#### 开启群老婆过滤本人
- **权限**: OWNER
- **用法**: `开启群老婆过滤本人`
- **描述**: 开启过滤本人和机器人账号

#### 关闭群老婆过滤本人
- **权限**: OWNER
- **用法**: `关闭群老婆过滤本人`
- **描述**: 关闭过滤本人和机器人账号

## WebUI管理

### 访问方式

1. 打开NapCat WebUI
2. 进入QTBOT管理面板
3. 点击左侧"群老婆设置"菜单

### 配置选项

#### 抽取模式
- **单群模式**: 仅从当前群成员中抽取
- **全群模式**: 从所有群的成员中抽取
- **注意**: 全群模式数据量大，群数多时可能较慢

#### 冷却时间
- **范围**: 1-3600秒
- **默认**: 30秒
- **说明**: 更换老婆的冷却时间，防止频繁更换

#### 过滤设置
- **过滤机器人**: 过滤掉机器人账号
- **过滤本人**: 过滤掉发送者和机器人本人账号

### 快捷指令

WebUI界面底部显示了常用的快捷指令，方便用户快速操作：
- `今日老婆` - 抽取今日老婆
- `更换老婆` - 更换老婆
- `群老婆` - 查看功能菜单

```json

  "123456789": 749590057,
  "234567890": 123456789
```

- **值**: 来源群号（"本群"或实际群号）

```json
{
  "123456789": 1772089127,
  "234567890": 1772089130
```

- **键**: 用户QQ号
- **值**: 最后更换时间（Unix时间戳）

### 核心逻辑

#### 1. 抽取老婆

```typescript
async function handleDrawWaifu(
  message: string,
  event: OB11Message,
  ctx: NapCatPluginContext,
  parsed: Record<string, string>
): Promise<void> {
  const userId = String(event.user_id);
  const today = getTodayDate();
  const mode = getWaifuMode();
  
  // 确定数据路径
  let dataPath = `筱筱吖/娱乐系统/今日老婆/群组模式/${mode}/`;
  if (mode === '单群模式') {
    dataPath += `${event.group_id}/`;
  }
  
  // 检查今日是否已抽取
  const todayWaifu = pluginState.loadDataFile(
    `${dataPath}${today}/记录.json`,
    userId,
    0
  );
  
  const command = message.match(/^(今日老婆|我的老婆|换群老婆|更换老婆)$/)?.[1];
  
  // 如果是今日老婆且已经抽取过
  if ((command === '今日老婆' || command === '我的老婆') && todayWaifu !== 0) {
    const userInfo = await ctx.actions.call('get_stranger_info', {
      user_id: todayWaifu
    }, ...);
    
    await sendReplyMessage(ctx, event, 组装消息);
    return;
  }
  
  // 检查冷却时间
  const cooldown = pluginState.loadDataFile('筱筱吖/娱乐系统/今日老婆/处理.json', '冷却', 30);
  const lastTime = pluginState.loadDataFile(`${dataPath}${today}/冷却.json`, userId, 0);
  const now = Math.floor(Date.now() / 1000);
  
  if (now - lastTime < cooldown && (command === '换群老婆' || command === '更换老婆')) {
    const remaining = cooldown - (now - lastTime);
    await sendReplyMessage(ctx, event, `冷却时间要${cooldown}秒哟～\n再等${remaining}秒再来吧～！`);
    return;
  }
  
  // 获取成员列表
  let members = [];
  if (mode === '全群模式') {
    const groups = await ctx.actions.call('get_group_list', {}, ...);
    for (const group of groups) {
      const groupMembers = await ctx.actions.call('get_group_member_list', {
        group_id: group.group_id
      }, ...);
      members = members.concat(groupMembers);
    }
  } else {
    members = await ctx.actions.call('get_group_member_list', {
      group_id: event.group_id
    }, ...);
  }
  
  // 过滤设置
  const filterRobot = pluginState.loadDataFile('筱筱吖/娱乐系统/今日老婆/处理.json', '过滤人机', '开启');
  const filterSelf = pluginState.loadDataFile('筱筱吖/娱乐系统/今日老婆/处理.json', '过滤本人', '开启');
  
  // 随机抽取
  const history = [yesterdayWaifu, todayWaifu];
  let maxRetry = 5;
  
  for (let i = 0; i < maxRetry; i++) {
    const randomIndex = Math.floor(Math.random() * members.length);
    const selected = members[randomIndex];
    
    // 检查重复
    if (history.includes(String(selected.user_id))) {
      continue;
    }
    
    // 过滤机器人
    if (filterRobot === '开启' && selected.is_robot) {
      continue;
    }
    
    // 过滤本人
    if (filterSelf === '开启' && (selected.user_id === event.self_id || selected.user_id === event.user_id)) {
      continue;
    }
    
    // 保存结果
    pluginState.saveDataFile(`${dataPath}${today}/记录.json`, userId, selected.user_id);
    pluginState.saveDataFile(`${dataPath}${today}/记录2.json`, userId, event.group_id);
    pluginState.saveDataFile(`${dataPath}${today}/冷却.json`, userId, now);
    
    // 发送结果
    await sendReplyMessage(ctx, event, 组装消息);
    return;
  }
  
  await sendReplyMessage(ctx, event, '我找了好几次都匹配不中唉？～要不待会再来？');
}
```

#### 3. 更改冷却时间

```typescript
async function handleChangeCooldown(
  message: string,
  event: OB11Message,
  ctx: NapCatPluginContext,
  parsed: Record<string, string>
): Promise<void> {
  const cooldown = Number(parsed.group0);
  
  if (cooldown < 1 || cooldown > 3600) {
    await sendReplyMessage(ctx, event, '冷却时间范围：1-3600秒');
    return;
  }
  
  pluginState.saveDataFile('筱筱吖/娱乐系统/今日老婆/处理.json', '冷却', cooldown);
  
  await sendReplyMessage(ctx, event, `好哒～！这就把设置群老婆的冷却改成 ${cooldown}秒！`);
}
```

## 注意事项

1. **授权要求**: 群老婆功能需要卡密授权
2. **全群模式性能**: 全群模式会遍历所有群的成员，群数多时可能较慢
3. **历史记录**: 只记录昨天和今天的老婆，避免重复
4. **冷却限制**: 更换老婆受冷却时间限制
5. **过滤设置**: 管理员指令对全局生效，不单针对某个群
6. **WebUI管理**: 建议使用WebUI管理界面，更直观方便
